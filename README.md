# terra-town

歩いた場所が開拓され、自分の街が育っていく GPS 位置情報ゲーム（Android 先行）。

## コンセプト

- GPS を用いてマップを歩く。**歩いた場所の霧が晴れて開拓される**
- 開拓エリアから資材を獲得し、実在地図の上に**自分の街を建設・育成**する
- **歩く楽しさ + 健康促進 + マップの開拓**を組み合わせる
- 行ったことがない場所も**ポイントで開拓**できる
- → 最終的に「**地球が自分の街になる**」楽しさを目指す
